Run my morning ritual for the BigB-PKM vault (`~/Documents/BigB-PKM`). Keep it FAST — under two minutes. Do the mechanical steps yourself; only ask me for the genuinely human parts. Never reformat or rewrite the rest of the note — only fill the sections named below, and preserve all existing content.

1. Run `today-note` (creates today's daily note if needed), then `rollover` (carries yesterday's unfinished tasks into today's `## Tasks`). Report in ONE line how many tasks rolled over.
2. Read today's note. In 2–3 lines max, reflect back: the date, the tasks now in `## Tasks`, and — only if you can find one quickly by scanning earlier `01 - Daily Notes/*.md` for the same month-day — one "on this day a year ago" note. Don't belabor it.
3. My Main 3 for today: **$ARGUMENTS** — if empty, ask: *"What are your Main 3 today? (1 essential, 1–2 urgent, rest is maintenance)"*. Write them as `- [ ] <item>` lines under the `### Main 3 …` heading, replacing the empty `- [ ]` placeholder there.
4. Ask one short line: *"Any prapti / intention to carry today?"* — if I answer, add it under `### My Prapti` as a `- ` bullet; if I skip, move on without fuss.
5. Close by confirming the note is set, and remind me in one line: capture tasks anytime with `ot "…"`, thoughts with SUPER+SHIFT+O, and close the day with `/end-of-day`. Do NOT open an editor.

If I'm clearly low-energy or in a hurry, just do steps 1–2 and stop — note + rollover alone is a complete, valid morning. A short day still counts.
